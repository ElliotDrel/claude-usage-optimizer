## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-04-01 | Tampering | app_meta JSON parse (schedule_fires) | mitigate | Wrap JSON.parse in try/catch; default to [] on parse failure; log [scheduler] corrupt schedule_fires, resetting |
| T-04-02 | Denial | catch-up loop at startup | accept | Bounded to schedule_fires array (max 5 items); runs once; not a DoS surface |
| T-04-03 | Tampering | timezone string from app_meta | mitigate | Use try/catch around Intl.DateTimeFormat construction; fall back to 'America/Los_Angeles' on error |
| T-04-04 | Denial | runTick errors stopping the interval | mitigate | runTick is wrapped in setInterval callback; any thrown error from runTick itself should be caught at the setInterval boundary — add top-level try/catch in the setInterval callback |
